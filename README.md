# 🍽️ Caixa Parmegiana

Sistema simples de caixa para pedidos de Filé à Parmegiana, feito em Python.

## 🎯 Objetivo
Registrar pedidos e evitar erros de caixa, com cálculo automático de valores.

## ⚙️ Funcionalidades
- Cadastro de pedidos
- Cálculo automático (individual, duplo, triplo)
- Extras (refrigerante e entrega)
- Relatório de caixa do dia

## 🛠️ Tecnologias
- Python
- pyinstaller (Para rodar em máquinas externas)
- Tkinter (Interface Gráfica)

## ▶️ Como rodar o projeto
Apenas executar o app.exe
