# 🍽️ Caixa Parmegiana

Sistema simples de caixa para pedidos de Filé à Parmegiana, feito em Python + Flask. (Assistenciado por IA)

 Objetivo
Registrar pedidos e evitar erros de caixa, com cálculo automático de valores.

 Funcionalidades
- Cadastro de pedidos
- Cálculo automático (individual, duplo, triplo)
- Extras (refrigerante e entrega)
- Relatório de caixa do dia

 Tecnologias
- Python
- Flask
- SQLite
- HTML/CSS

  Como rodar o projeto
```bash
pip install -r requirements.txt
python app.py
